## Quickstart

Requirements:

- Go 1.26.1 or later

1. Clone the repo

    ```bash
    git clone https://github.com/BananaJeanss/go-ship.git
    ```

2. cd into the project directory

    ```bash
    cd go-ship
    ```

3. Install dependencies

    ```bash
    go mod tidy
    ```

4. Run the server

    ```bash
    task run
    ```

5. Open your browser and navigate to `http://localhost:3000`

## Taskfile

- `task run`: go run .
- `task build`: go build with flags
- `task test`: go test ./...

Requirements

  • Project must be written using Go
  • Open-source your code on GitHub, GitLab, or anywhere else!
  • Must be your original work.
    Feel free to take inspiration from existing projects, but don't copy-paste!
  • Write a proper README.md with instructions on how to build and run your project.
    AI will not be allowed for this.
  • AI generation can only be used for up to 30% of your code.
  • Ship your project!
    • Websites: Must be live and accessible to the public
    • Anything else: Upload the binary to GitHub Releases (or another platform),
      and optionally list your package on pkg.go.dev!

Frequently Asked Questions

What even is Go?

Go (aka Golang) is a high-level general-purpose programming language released in 2009 by Google.
What makes it special are its super-fast compile times, concurrency via goroutines, simplicity & efficiency, and its powerful standard library.
Compared to C++ or Rust, Go is much easier to learn and write, while still offering comparable performance for 99% of use cases.
Some popular projects written in Go are Caddy, Docker, Kubernetes, and Dgraph.

If you're 13-18 years old, yes!

How do I ship my project?

If your project is a website, you can host it on any platform that supports Go (try Hack Club Nest! (if it's back up yet)) as long as it's live and accessible to the public.
If your project is not a website, you can upload the binary to GitHub Releases (or another platform) and optionally list your package on pkg.go.dev.
